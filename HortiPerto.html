Cadastro de Vendedor

Requisitos para Cadastro

  • Cadastro de Produtor Rural (CPP - Cadastro de Produtor Primário)
  • Inscrição Estadual
  • Capacidade para emissão de Nota Fiscal de Produtor Rural
  • Cadastro Ambiental Rural (CAR) para propriedades rurais
  • Alvará Sanitário (apenas para produtos processados)

A inscrição no CAR é obrigatória para todos os imóveis rurais do país, constitui-se no primeiro passo para a regularização ambiental e dá acesso a benefícios previstos no Código Florestal (Lei nº 12.651/2012).
